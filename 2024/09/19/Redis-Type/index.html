Hash

HashжҳҜдёҖдёӘй”®еҖјеҜ№зҡ„йӣҶеҗҲ

Hash жҳҜдёҖдёӘ String зұ»еһӢзҡ„ fieldпјҲеӯ—ж®өпјү е’Ң valueпјҲеҖјпјү зҡ„жҳ е°„иЎЁпјҢhash зү№еҲ«йҖӮеҗҲз”ЁдәҺеӯҳеӮЁеҜ№иұЎгҖӮ

Hashзұ»еһӢдёҺpythonдёӯеӯ—е…ё(й”®еҖјеҜ№)зұ»еһӢзҡ„ж•°жҚ®жҜ”иҫғзӣёдјјпјҢеҸӘдёҚиҝҮhashзұ»еһӢдёӯдҪҝз”Ёзҡ„жҳҜfieldе’ҢvalueпјҢеӯ—е…ёдёӯдҪҝз”Ёзҡ„жҳҜkeyе’Ңvalue

еңЁеӯ—е…ёдёӯжҹҘиҜўдёҖдёӘvalueеҫҖеҫҖйңҖиҰҒжҢҮе®ҡkeyпјҢиҖҢеңЁredisдёӯжҹҘиҜўдёҖдёӘfiledйңҖиҰҒеҗҢж—¶жҢҮе®ҡkeyе’Ңfiled

Zset

ZsetдёҺSetйқһеёёзӣёдјјпјҢжҳҜдёҖдёӘжІЎжңүйҮҚеӨҚе…ғзҙ зҡ„StringйӣҶеҗҲгҖӮдёҚеҗҢд№ӢеӨ„жҳҜZsetзҡ„жҜҸдёӘе…ғзҙ йғҪе…іиҒ”дәҶдёҖдёӘеҲҶж•°

ZsetеҸҜд»ҘйҖҡиҝҮдёҖдёӘеҲҶж•°жқҘеҜ№е…ғзҙ иҝӣиЎҢжҺ’еәҸпјҢжҳҜжңүеәҸйӣҶеҗҲпјҢиҝҷдёӘеҲҶж•°еҶіе®ҡдәҶZestдёӯе…ғзҙ зҡ„зҙўеј•дҪҚзҪ®

пјҲе’Ңhashиҝҷз§Қеӯ—е…ёз»“жһ„зҡ„дёҚеҗҢпјҢд»–еҸӘиҰҒеҫҖйҮҢеҠ е…ғзҙ е°ұиЎҢпјү

Bitmaps

жҳҜдёҖз§ҚдәҢиҝӣеҲ¶зҡ„еӯҳеӮЁж–№ејҸпјҢеҸӘжңү0е’Ң1

RedisжҸҗдҫӣдәҶBitmapsиҝҷдёӘ вҖңж•°жҚ®з»“жһ„вҖқ еҸҜд»Ҙе®һзҺ°еҜ№дҪҚзҡ„ж“ҚдҪң
еҗҲзҗҶең°дҪҝз”ЁдҪҚиғҪеӨҹжңүж•Ҳең°жҸҗй«ҳеҶ…еӯҳдҪҝз”ЁзҺҮе’ҢејҖеҸ‘ж•ҲзҺҮгҖӮ

Geospatiaзұ»еһӢ-ең°зҗҶдҝЎжҒҜ

GEOпјҢGeographic,ең°зҗҶдҝЎжҒҜзҡ„зј©еҶҷгҖӮиҜҘзұ»еһӢе°ұжҳҜе…ғзҙ зҡ„дәҢз»ҙеқҗж ҮпјҢеңЁең°еӣҫдёҠе°ұжҳҜз»Ҹзә¬еәҰгҖӮRedisеҹәдәҺиҜҘзұ»еһӢпјҢжҸҗдҫӣдәҶз»Ҹзә¬еәҰи®ҫзҪ®гҖҒжҹҘиҜўгҖҒиҢғеӣҙжҹҘиҜўгҖҒи·қзҰ»жҹҘиҜўгҖҒз»Ҹзә¬еәҰHashзӯүеёёи§Ғж“ҚдҪңгҖӮ

HyperloglogвҖ”вҖ”жөҒйҮҸз»ҹи®ЎгҖҒж’ӯж”ҫж•°зӯү

еҒҡз«ҷзӮ№жөҒйҮҸз»ҹи®Ўзҡ„ж—¶еҖҷдёҖиҲ¬дјҡз»ҹи®ЎйЎөйқўUV(зӢ¬з«Ӣи®ҝе®ў:unique visitor)е’ҢPV(еҚійЎөйқўжөҸи§ҲйҮҸпјҡpage view)гҖӮredis HyperLogLogжҳҜз”ЁжқҘеҒҡеҹәж•°з»ҹи®Ўзҡ„з®—жі•пјҢHyperLogLogзҡ„дјҳзӮ№жҳҜпјҡеңЁиҫ“е…Ҙе…ғзҙ зҡ„ж•°йҮҸжҲ–иҖ…дҪ“з§ҜйқһеёёйқһеёёеӨ§ж—¶пјҢи®Ўз®—еҹәж•°жүҖйңҖзҡ„з©әй—ҙжҖ»жҳҜеӣәе®ҡе№¶дё”еҫҲе°Ҹзҡ„
